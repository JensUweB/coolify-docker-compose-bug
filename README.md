# Example Repository for Coolify Bug
This is an example repo to easier reproduce the [Coolify issue 3517](https://github.com/coollabsio/coolify/issues/3517)

## Steps to reproduce:
1. Fork this repository
2. Make your forked repo private
3. Add new Application to Coolify with "Private Repository (Deploy key)" option
4. Set Build Pack to "Docker Compose", set Docker Compose Location to `/docker/docker-compose.yaml` & hit save.
